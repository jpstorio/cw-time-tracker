## Project Setup

1. Clone the repository:

```sh
git clone https://github.com/jpstorio/cw-time-tracker.git
```

2. Navigate to the project directory:

```sh
cd cw-time-tracker
```

3. Install the package:

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
